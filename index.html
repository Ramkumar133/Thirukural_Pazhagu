Services

Web Based Game

It is a web based application. It has two modules learning and gaming section where the user can learn kural and then they can play it to test their knowledge.

Responsive Animation

Everything in the website will have feel good animation which will please all the users while using the website.

User Progress

The progress of each user is maintained by displaying the streaks , kurals completed, scoreboard, leaderboard in the dashboard.

About Project

  • Problem Statement

    Student all over world find very difficult to learn Thirukkural without guide. Learning Thirukkural at younger stage helps them to live peacefully.

  • Our objectives

    To help the children to gain knowledge and correct pronunciation of kurals.
    To have games with different difficulty levels, so that new user can easily adapt to it

  • Impact on Environment

    This mainly focus on expressing the meaning of each kural and how they can be used in daily life. After completing each level students gain the knowledge of each kural and enrich their character. It also improves the ethical way of living.

  • Modules

    1. Learning Section
    2. Listening and Practicing
    3. Playing Games
    4. Database Connection
    5. Score Calculation
